HTML CHEETSHEET

Basic Tags

Tag Name Description
<html> HTML Creates an HTML document
<head> head Sets off the title and other info that isn't displayed.
<body> Body Sets off the visible portion of the document.
<title> Title Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked.

Text Tags

Tags Name Description
<h1> Heading Creates headlines--H1=largest, H6=smallest
<b> Bold Creates a bold text
<i> Italic Creates a italic text
<em> Em Emphasizes a word, usually processed as italics
<strong> Strong Emphasizes a word, usually processed as bold
<p> Paragraph Creates a new paragraph
<br> Line Break Inserts a line break
<div> div Used to format block content with css
<span> Span Used to format inline content with css

Lists

Tags Name Description
<ul> Unordered List Creates an unordered list
<ol> Ordered List Creates an ordered list
<li> List Item Emcompasses each list item

Table Tags

Tag Name Description
<table> Table The wrapper element for all html tables
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containig actual table data.
<tr> Table Row The table row container.
<td> Table data The table data container.
<th> Table Heading The table heading container.

Table Attributes

Tags Name Description
colspan Column Span Defines how many columns a td elment should span.
rowspan Row span Defines how many rows a td element should span
